§4 三重积分

4.1 三重积分的概念与性质

定义（三重积分）

三重积分是二重积分在三维空间中的直接推广。对 \(\Omega \subset \mathbb{R}^{3}\) 上的函数 \(f(x, y, z)\)，通过分割、求和、取极限的方式定义：

\(\iiint_{\Omega} f(x, y, z)\,dx\,dy\,dz\)

其性质（线性性、区域可加性、保序性、中值定理等）与二重积分类似。

4.2 三重积分的计算

定理 4.1（投影法/先一后二）

设 \(\Omega\) 在 \(xOy\) 平面上的投影为有界闭区域 \(D\)，且 \(\Omega\) 可表示为

\(\Omega = \{(x, y, z) \mid (x, y) \in D,\; z_{1}(x, y) \le z \le z_{2}(x, y)\}\)

则

\(\iiint_{\Omega} f(x, y, z)\,dx\,dy\,dz = \iint_{D} \left[\int_{z_{1}(x, y)}^{z_{2}(x, y)} f(x, y, z)\,dz\right] dx\,dy\)

定理 4.2（截面法/先二后一）

设 \(\Omega\) 在 \(z\) 轴上的投影区间为 \([c, d]\)，对每个 \(z \in [c, d]\)，用垂直于 \(z\) 轴的平面截 \(\Omega\) 得截面 \(D_{z}\)，则

\(\iiint_{\Omega} f(x, y, z)\,dx\,dy\,dz = \int_{c}^{d} \left[\iint_{D_{z}} f(x, y, z)\,dx\,dy\right] dz\)

例

计算 \(I = \iiint_{\Omega} y(x+z)\,dx\,dy\,dz\)，\(\Omega\) 由 \(y=0, z=0, x+z=\pi/2, y=x\) 围成。

解：\(\Omega\) 在 \(xOy\) 上的投影 \(D: y=0, y=x, x=\pi/2\)。

\(I = \iint_{D} dx\,dy \int_{0}^{\pi/2-x} y(x+z)\,dz = \frac{\pi^{4}}{256}\)

例（截面法）

计算 \(I = \iiint_{\Omega} z^{2}\,dx\,dy\,dz\)，\(\Omega\) 为锥面 \(z^{2}=x^{2}+y^{2}\) 与平面 \(z=1\) 围成的区域。

解：用截面法，对每个 \(z \in [0,1]\)，截面 \(D_{z}: x^{2}+y^{2} \le z^{2}\)，面积为 \(\pi z^{2}\)。

\(I = \int_{0}^{1} z^{2} \left(\iint_{D_{z}} dx\,dy\right) dz = \int_{0}^{1} z^{2} \cdot \pi z^{2}\,dz = \frac{\pi}{5}\)

4.3 三重积分的对称性

对称性

若 \(\Omega\) 关于 \(xOy\) 平面对称，且 \(f\) 关于 \(z\) 为奇函数（\(f(x,y,-z) = -f(x,y,z)\)），则 \(\iiint_{\Omega} f = 0\)；若 \(f\) 关于 \(z\) 为偶函数，则积分等于上半区域积分值的2倍。

例

计算 \(I = \iiint_{\Omega} (x+y+z)^{2}\,dx\,dy\,dz\)，\(\Omega: \frac{x^{2}}{a^{2}}+\frac{y^{2}}{b^{2}}+\frac{z^{2}}{c^{2}} \le 1\)。

解：展开得 \((x+y+z)^{2} = x^{2}+y^{2}+z^{2}+2xy+2yz+2zx\)。由对称性，交叉项积分为0。故

\(I = \iiint_{\Omega} (x^{2}+y^{2}+z^{2})\,dx\,dy\,dz\)

由对称性，\(\iiint x^{2} = \iiint y^{2} = \iiint z^{2} = \frac{1}{3}\iiint (x^{2}+y^{2}+z^{2})\)。利用广义球坐标计算得

\(I = \frac{4\pi}{15}abc(a^{2}+b^{2}+c^{2})\)

4.4 柱坐标与球坐标

柱坐标变换

\(\begin{cases} x = r\cos\theta, \\ y = r\sin\theta, \\ z = z, \end{cases} \quad \frac{\partial(x, y, z)}{\partial(r, \theta, z)} = r\)
\(\iiint_{\Omega} f(x, y, z)\,dx\,dy\,dz = \iiint_{\Omega^{*}} f(r\cos\theta, r\sin\theta, z)\,r\,dr\,d\theta\,dz\)

球坐标变换

\(\begin{cases} x = \rho\sin\varphi\cos\theta, \\ y = \rho\sin\varphi\sin\theta, \\ z = \rho\cos\varphi, \end{cases} \quad \frac{\partial(x, y, z)}{\partial(\rho, \varphi, \theta)} = \rho^{2}\sin\varphi\)
\(\iiint_{\Omega} f(x, y, z)\,dx\,dy\,dz = \iiint_{\Omega^{*}} f(\rho\sin\varphi\cos\theta, \rho\sin\varphi\sin\theta, \rho\cos\varphi)\,\rho^{2}\sin\varphi\,d\rho\,d\varphi\,d\theta\)

其中 \(\rho \ge 0,\; 0 \le \varphi \le \pi,\; 0 \le \theta \le 2\pi\)。

广义球坐标

对椭球区域，作广义球坐标变换：

\(\begin{cases} x = a\rho\sin\varphi\cos\theta, \\ y = b\rho\sin\varphi\sin\theta, \\ z = c\rho\cos\varphi, \end{cases} \quad \frac{\partial(x, y, z)}{\partial(\rho, \varphi, \theta)} = abc\,\rho^{2}\sin\varphi\)

例（柱坐标）

\(I = \iiint_{\Omega} (x^{2}+y^{2})\,dx\,dy\,dz\)，\(\Omega: x^{2}+y^{2} \le z \le 1\)。

解：用柱坐标，\(\Omega^{*}: 0 \le r \le 1,\; 0 \le \theta \le 2\pi,\; r^{2} \le z \le 1\)。

\(I = \int_{0}^{2\pi} d\theta \int_{0}^{1} r^{3}\,dr \int_{r^{2}}^{1} dz = \frac{\pi}{6}\)

例（球坐标）

计算 \(I = \iiint_{\Omega} ze^{-(x^{2}+y^{2}+z^{2})}\,dx\,dy\,dz\)，\(\Omega\) 为锥面 \(z=\sqrt{x^{2}+y^{2}}\) 与球面 \(x^{2}+y^{2}+z^{2}=1\) 围成的区域。

解：用球坐标，\(\Omega\) 对应于 \(0 \le \rho \le 1,\; 0 \le \varphi \le \pi/4,\; 0 \le \theta \le 2\pi\)。

\(I = \int_{0}^{2\pi} d\theta \int_{0}^{\pi/4} \cos\varphi\sin\varphi\,d\varphi \int_{0}^{1} \rho^{3}e^{-\rho^{2}}\,d\rho = \frac{\pi}{2}\left(\frac{1}{2} - \frac{1}{e}\right)\)

例（椭球体积）

椭球 \(\frac{x^{2}}{a^{2}}+\frac{y^{2}}{b^{2}}+\frac{z^{2}}{c^{2}} \le 1\) 的体积：

方法一（广义球坐标）：

\(V = \iiint_{\Omega} dx\,dy\,dz = abc \int_{0}^{2\pi} d\theta \int_{0}^{\pi} \sin\varphi\,d\varphi \int_{0}^{1} \rho^{2}\,d\rho = \frac{4\pi}{3}abc\)

方法二（截面法）：

\(V = \int_{-c}^{c} \pi ab\left(1-\frac{z^{2}}{c^{2}}\right) dz = \frac{4\pi}{3}abc\)
