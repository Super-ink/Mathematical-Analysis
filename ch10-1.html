§1 无穷级数的基本概念

定义（无穷级数）

设 \(\{a_{n}\}\) 是一个序列（数列）。我们称和式

\(a_{1} + a_{2} + \cdots + a_{n} + \cdots\)

为无穷级数（或数项级数），简写为

\(\sum\limits_{n=1}^{+\infty} a_{n}\)

其中 \(a_{n}\) 称为级数的通项，\(S_{n} = \sum\limits_{k=1}^{n} a_{k}\) 称为级数的部分和。

定义（级数的收敛与发散）

若级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 的部分和序列 \(\{S_{n}\}\) 当 \(n \to +\infty\) 时有极限，则称级数 收敛，否则称级数 发散。

当级数收敛时，称

\(S = \lim\limits_{n\to+\infty} S_{n} = \sum\limits_{n=1}^{+\infty} a_{n}\)

为级数的和。

注

当级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 收敛时，不严格区分"级数"与"级数的和"两个概念。

定理 1.1（收敛的必要条件）

若 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 收敛，则

\(\lim\limits_{n\to+\infty} a_{n} = 0\)

注

\(\lim\limits_{n\to+\infty} a_{n} = 0\) 只是级数收敛的必要条件，而非充分条件。

定理 1.2（Cauchy收敛原理）

级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 收敛的充要条件是：

\(\forall \varepsilon > 0,\; \exists N \in \mathbb{N},\; \forall n \ge N,\; \forall p \in \mathbb{N},\; |S_{n+p} - S_{n}| = \bigl|a_{n+1} + a_{n+2} + \cdots + a_{n+p}\bigr| < \varepsilon\)

即部分和序列 \(\{S_{n}\}\) 是 Cauchy 列。

例

（几何级数） \(\sum\limits_{n=1}^{+\infty} q^{n-1}\)，当 \(|q| < 1\) 时收敛到 \(\frac{1}{1-q}\)；当 \(|q| \ge 1\) 时发散。

例

（调和级数） \(\sum\limits_{n=1}^{+\infty} \frac{1}{n}\) 发散。因为

\(S_{2n} - S_{n} = \frac{1}{n+1} + \cdots + \frac{1}{2n} \ge n \cdot \frac{1}{2n} = \frac{1}{2}\)

不满足 Cauchy 收敛原理。

例

（p-级数） \(\sum\limits_{n=1}^{+\infty} \frac{1}{n^{2}}\) 收敛。由 Cauchy 收敛原理：

\(|S_{n+p} - S_{n}| = \frac{1}{(n+1)^{2}} + \cdots + \frac{1}{(n+p)^{2}} < \frac{1}{n(n+1)} + \cdots + \frac{1}{(n+p-1)(n+p)} = \frac{1}{n} - \frac{1}{n+p} < \frac{1}{n}\)

定理 1.3（线性性质）

若 \(\sum\limits_{n=1}^{+\infty} a_{n} = A\)，\(\sum\limits_{n=1}^{+\infty} b_{n} = B\)，则对任意常数 \(\alpha, \beta\)，有

\(\sum\limits_{n=1}^{+\infty} (\alpha a_{n} + \beta b_{n}) = \alpha A + \beta B\)

即收敛级数可逐项相加。

定理 1.4（改变有限项不影响收敛性）

在级数中去掉、添加或改变有限项，不改变级数的收敛性。

定理 1.5（加括号与收敛性）

若级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 收敛，则对其任意加括号后得到的级数仍收敛，且和不变。即收敛级数满足结合律。

注

加括号后收敛不能推出原级数收敛。例如：\(1-1+1-1+\cdots\) 发散，但加括号为 \((1-1)+(1-1)+\cdots = 0\) 却收敛。

定理 1.6

若加括号后得到的级数收敛，且加括号时同一括号内各项的符号相同（或通项趋于 0），则原级数也收敛。
