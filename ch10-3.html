§3 任意项级数

3.1 交错级数

定义（交错级数）

级数 \(\sum\limits_{n=1}^{+\infty} (-1)^{n+1} a_{n}\)（\(a_{n} > 0\)）称为交错级数。

定义（Leibniz莱布尼茨级数）

若交错级数 \(\sum\limits_{n=1}^{+\infty} (-1)^{n+1} a_{n}\) 满足：

(1) \(a_{n+1} \le a_{n}\)（\(n = 1, 2, \ldots\)）

(2) \(\lim\limits_{n\to+\infty} a_{n} = 0\)

则称此级数为Leibniz级数。

定理 3.1（Leibniz莱布尼茨判别法）

Leibniz级数必收敛。

注 1

Leibniz级数 \(\sum\limits_{n=1}^{+\infty} (-1)^{n+1} a_{n}\) 的余和 \(r_{n} = \sum\limits_{k=n+1}^{+\infty} (-1)^{k+1} a_{k}\) 的符号与首项 \((-1)^{n+2} a_{n+1}\) 的符号相同，且有 \(|r_{n}| \le a_{n+1}\)。

3.2 绝对收敛与条件收敛

定义（绝对收敛与条件收敛）

若 \(\sum\limits_{n=1}^{+\infty} |a_{n}|\) 收敛，则称 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 绝对收敛。

若 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 收敛，但 \(\sum\limits_{n=1}^{+\infty} |a_{n}|\) 发散，则称 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 条件收敛。

3.3 Abel阿贝尔判别法与Dirichlet狄利克雷判别法

引理 3.1（Abel阿贝尔变换/分部求和公式）

设 \(\{a_{n}\}, \{b_{n}\}\) 是两个数列，记 \(B_{n} = \sum\limits_{i=1}^{n} b_{i}\)（\(n = 1, 2, \ldots\)），则

\(\sum\limits_{i=1}^{n} a_{i}b_{i} = a_{n}B_{n} - \sum\limits_{i=1}^{n-1} (a_{i+1} - a_{i})B_{i}\)

注

Abel变换是离散形式的分部积分公式，故亦称为分部求和公式。

引理 3.2（Abel阿贝尔引理）

设

(1) \(\{a_{n}\}\) 为单调数列，

(2) \(\{B_{n}\}\) 为有界数列，即存在 \(M > 0\)，使得 \(\forall n \in \mathbb{N}, |B_{n}| \le M\)。

则

\(\bigl|\sum\limits_{i=1}^{n} a_{i}b_{i}\bigr| \le M(|a_{1}| + 2|a_{n}|)\)

定理 3.2（Dirichlet狄利克雷判别法）

若 \(\{a_{n}\}\) 单调且 \(\lim\limits_{n\to+\infty} a_{n} = 0\)，部分和\(\{\sum\limits_{i=1}^{n} b_{i}\}\) 有界，则 \(\sum\limits_{n=1}^{+\infty} a_{n}b_{n}\) 收敛。

注

Leibniz判别法是Dirichlet判别法的特例。

例（Dirichlet判别法的应用）

设 \(\{a_{n}\}\) 单调趋于 0，则级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\sin nx\) 和 \(\sum\limits_{n=1}^{+\infty} a_{n}\cos nx\) 当 \(x \neq 2k\pi\) 时收敛。

这是因为 \(\sum\limits_{k=1}^{n} \sin kx = \frac{\cos\frac{x}{2} - \cos(n+\frac{1}{2})x}{2\sin\frac{x}{2}}\) 有界，由Dirichlet判别法即得。

定理 3.3（Abel阿贝尔判别法）

若 \(\{a_{n}\}\) 单调有界，\(\sum\limits_{n=1}^{+\infty} b_{n}\) 收敛，则 \(\sum\limits_{n=1}^{+\infty} a_{n}b_{n}\) 收敛。

3.4 绝对收敛级数与条件收敛级数的性质

命题 3.1

设 \(a_{n}^{+} = \frac{|a_{n}| + a_{n}}{2}\)，\(a_{n}^{-} = \frac{|a_{n}| - a_{n}}{2}\)。

(1) \(\sum\limits_{n=1}^{+\infty} a_{n}\) 绝对收敛的充要条件是：\(\sum\limits_{n=1}^{+\infty} a_{n}^{+}\) 和 \(\sum\limits_{n=1}^{+\infty} a_{n}^{-}\) 同时收敛；

(2) 若 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 条件收敛，则 \(\sum\limits_{n=1}^{+\infty} a_{n}^{+}\) 和 \(\sum\limits_{n=1}^{+\infty} a_{n}^{-}\) 同时发散。但反之不真。

定义（重排级数/更序级数）

给定级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\)，设 \(\gamma: \mathbb{N} \to \mathbb{N}\) 是双射。则 \(\sum\limits_{n=1}^{+\infty} a_{\gamma(n)}\) 称为 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 的一个重排级数（或更序级数）。

定理 3.4

若 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 绝对收敛，则它的任何一个重排级数也绝对收敛，且

\(\sum\limits_{n=1}^{+\infty} a_{\gamma(n)} = \sum\limits_{n=1}^{+\infty} a_{n}\)

注

上述定理说明对绝对收敛的无穷级数，加法交换律成立。

定理 3.5（Riemann黎曼重排定理）

若 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 条件收敛，则 \(\forall A \in [-\infty, +\infty]\)，都存在 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 的一个重排 \(\gamma\) 满足

\(\sum\limits_{n=1}^{+\infty} a_{\gamma(n)} = A\)

注

在上述定理中，当 \(A = \pm\infty\) 时，\(\sum\limits_{n=1}^{+\infty} a_{\gamma(n)}\) 发散。

例（Riemann重排）

考虑条件收敛的交错调和级数 \(\sum\limits_{n=1}^{+\infty} \frac{(-1)^{n+1}}{n} = \ln 2\)。将其重排为

\(1 - \frac{1}{2} - \frac{1}{4} + \frac{1}{3} - \frac{1}{6} - \frac{1}{8} + \frac{1}{5} - \frac{1}{10} - \frac{1}{12} + \cdots\)

即每个正项后跟两个负项，则新级数的和为 \(\frac{1}{2}\ln 2\)。

3.5 级数的乘法

命题 3.2

给定收敛级数 \(\sum\limits_{n=1}^{+\infty} a_{n}, \sum\limits_{n=1}^{+\infty} b_{n}\)，令

\(d_{n} = a_{1}b_{n} + a_{2}b_{n} + \cdots + a_{n}b_{n} + a_{n}b_{n-1} + \cdots + a_{n}b_{1}\)

则 \(\sum\limits_{n=1}^{+\infty} d_{n}\) 收敛，且

\(\sum\limits_{n=1}^{+\infty} d_{n} = \bigl(\sum\limits_{n=1}^{+\infty} a_{n}\bigr)\bigl(\sum\limits_{n=1}^{+\infty} b_{n}\bigr)\)

定义（Cauchy乘积）

称级数 \(\sum\limits_{n=1}^{+\infty} c_{n}\) 为级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 和 \(\sum\limits_{n=1}^{+\infty} b_{n}\) 的Cauchy乘积，其中

\(c_{n} = \sum\limits_{i+j=n+1} a_{i}b_{j} = a_{1}b_{n} + a_{2}b_{n-1} + \cdots + a_{n}b_{1}\)

注

级数 \(\sum\limits_{n=1}^{+\infty} a_{n}, \sum\limits_{n=1}^{+\infty} b_{n}\) 的收敛性不能保证其Cauchy乘积的收敛性。

定理 3.6

若级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 与 \(\sum\limits_{n=1}^{+\infty} b_{n}\) 都绝对收敛，则由 \(\{a_{i}b_{j}\}\) 构成的级数以任意方式排列求和都绝对收敛，且其和为 \((\sum\limits_{n=1}^{+\infty} a_{n})(\sum\limits_{n=1}^{+\infty} b_{n})\)。

注 1

上述定理说明绝对收敛的级数满足乘法的分配律。

注 2

上述定理说明两个绝对收敛的级数的Cauchy乘积绝对收敛。

定理 3.7（Mertens定理）

若级数 \(\sum\limits_{n=1}^{+\infty} a_{n}\) 绝对收敛，\(\sum\limits_{n=1}^{+\infty} b_{n}\) 收敛，则它们的Cauchy乘积 \(\sum\limits_{n=1}^{+\infty} c_{n}\) 也收敛，且满足

\(\sum\limits_{n=1}^{+\infty} c_{n} = \bigl(\sum\limits_{n=1}^{+\infty} a_{n}\bigr)\bigl(\sum\limits_{n=1}^{+\infty} b_{n}\bigr)\)

定理 3.8（Abel定理）

若级数 \(\sum\limits_{n=1}^{+\infty} a_{n}, \sum\limits_{n=1}^{+\infty} b_{n}\) 收敛，且它们的Cauchy乘积 \(\sum\limits_{n=1}^{+\infty} c_{n}\) 也收敛，则

\(\sum\limits_{n=1}^{+\infty} c_{n} = \bigl(\sum\limits_{n=1}^{+\infty} a_{n}\bigr)\bigl(\sum\limits_{n=1}^{+\infty} b_{n}\bigr)\)
