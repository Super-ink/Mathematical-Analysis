§12 空间曲线与曲面的几何

空间曲线

定义（空间曲线的参数方程）

空间曲线可以参数方程表示为

\(\Gamma:\; \boldsymbol{r} = \boldsymbol{r}(t) = (x(t), y(t), z(t)),\quad t \in I\)

其中 \(t\) 为参数。曲线的切向量为

\(\dot{\boldsymbol{r}}(t) = \frac{d\boldsymbol{r}}{dt} = (\dot{x}(t), \dot{y}(t), \dot{z}(t))\)

若 \(\dot{\boldsymbol{r}}(t) \neq 0\)，则称曲线在该点为正则点。曲线在 \(t_{0}\) 处的切线方程为

\(\boldsymbol{r} = \boldsymbol{r}(t_{0}) + \dot{\boldsymbol{r}}(t_{0})(t - t_{0})\)

或写为对称式：

\(\frac{x - x(t_{0})}{\dot{x}(t_{0})} = \frac{y - y(t_{0})}{\dot{y}(t_{0})} = \frac{z - z(t_{0})}{\dot{z}(t_{0})}\)

定义（法平面）

过曲线上的点 \(P_{0}\) 且垂直于切线的平面称为曲线的法平面，其方程为

\(\dot{x}(t_{0})(x - x(t_{0})) + \dot{y}(t_{0})(y - y(t_{0})) + \dot{z}(t_{0})(z - z(t_{0})) = 0\)

或写作向量形式 \((\boldsymbol{r} - \boldsymbol{r}(t_{0}), \dot{\boldsymbol{r}}(t_{0})) = 0\)。

定义（曲线的弧长）

设曲线 \(\boldsymbol{r} = \boldsymbol{r}(t)\) 在 \([a,b]\) 上 \(C^{1}\)，则从 \(t=a\) 到 \(t=b\) 的弧长为

\(s = \int_{a}^{b} \|\dot{\boldsymbol{r}}(t)\|\,dt = \int_{a}^{b} \sqrt{[\dot{x}(t)]^{2} + [\dot{y}(t)]^{2} + [\dot{z}(t)]^{2}}\,dt\)

弧微分：\(ds = \|\dot{\boldsymbol{r}}(t)\|\,dt = \sqrt{dx^{2} + dy^{2} + dz^{2}}\)。

例 1（圆锥螺线）

求曲线 \(\boldsymbol{r}(t) = (at\cos t, -at\sin t, bt)\)（\(a,b > 0\)）在 \(t \in [0, 2\pi]\) 上的弧长。

解：\(\dot{\boldsymbol{r}}(t) = (a\cos t - at\sin t,\; -a\sin t - at\cos t,\; b)\)

\(\|\dot{\boldsymbol{r}}(t)\|^{2} = a^{2}[\cos t - t\sin t]^{2} + a^{2}[-\sin t - t\cos t]^{2} + b^{2} = a^{2}t^{2} + a^{2} + b^{2}\)
\(s = \int_{0}^{2\pi} \sqrt{a^{2}t^{2} + a^{2} + b^{2}}\,dt = \frac{a}{2}\left[t\sqrt{t^{2}+c^{2}} + c^{2}\ln\left(t + \sqrt{t^{2}+c^{2}}\right)\right]_{0}^{2\pi}\)

其中 \(c^{2} = 1 + \frac{b^{2}}{a^{2}}\)。

空间曲面

定义（空间曲面的参数方程）

空间曲面可以参数方程表示为

\(\boldsymbol{r} = \boldsymbol{r}(u,v) = (x(u,v), y(u,v), z(u,v)),\quad (u,v) \in D\)

曲面的切向量为偏导向量：

\(\boldsymbol{r}_{u} = (x_{u}, y_{u}, z_{u}),\quad \boldsymbol{r}_{v} = (x_{v}, y_{v}, z_{v})\)

若 \(\boldsymbol{r}_{u} \times \boldsymbol{r}_{v} \neq 0\)，则称该点为正则点。曲面的法向量为

\(\boldsymbol{n}(u,v) = \boldsymbol{r}_{u} \times \boldsymbol{r}_{v} = \begin{vmatrix} \boldsymbol{i} & \boldsymbol{j} & \boldsymbol{k} \\ x_{u} & y_{u} & z_{u} \\ x_{v} & y_{v} & z_{v} \end{vmatrix} = \left(\frac{\partial(y,z)}{\partial(u,v)},\; \frac{\partial(z,x)}{\partial(u,v)},\; \frac{\partial(x,y)}{\partial(u,v)}\right)\)

定义（曲面的切平面与法线）

曲面 \(\boldsymbol{r} = \boldsymbol{r}(u,v)\) 在点 \(P_{0} = \boldsymbol{r}(u_{0},v_{0})\) 处的切平面方程为

\((\boldsymbol{r} - \boldsymbol{r}(u_{0},v_{0}),\; \boldsymbol{n}(u_{0},v_{0})) = 0\)

即：

\(\frac{\partial(y,z)}{\partial(u,v)}(P_{0})(x - x_{0}) + \frac{\partial(z,x)}{\partial(u,v)}(P_{0})(y - y_{0}) + \frac{\partial(x,y)}{\partial(u,v)}(P_{0})(z - z_{0}) = 0\)

法线方程为：

\(\frac{x - x_{0}}{\frac{\partial(y,z)}{\partial(u,v)}(P_{0})} = \frac{y - y_{0}}{\frac{\partial(z,x)}{\partial(u,v)}(P_{0})} = \frac{z - z_{0}}{\frac{\partial(x,y)}{\partial(u,v)}(P_{0})}\)

例 2（隐式曲面的切平面与法线）

求曲面 \(e^{z} - z + xy = 3\) 在点 \((2,1,0)\) 处的切平面和法线。

解：令 \(F(x,y,z) = e^{z} - z + xy - 3 = 0\)。则

\(F_{x} = y,\; F_{y} = x,\; F_{z} = e^{z} - 1\)

在 \((2,1,0)\) 处，\(\boldsymbol{n} = (F_{x}, F_{y}, F_{z}) = (1, 2, 0)\)。

切平面方程：\(1\cdot(x-2) + 2\cdot(y-1) + 0\cdot(z-0) = 0\)，即 \(x + 2y - 4 = 0\)。

法线方程：\(\frac{x-2}{1} = \frac{y-1}{2} = \frac{z}{0}\)（即 \(z = 0\) 平面上的直线）。

注（隐式曲面的法向量）

对于隐式方程 \(F(x,y,z) = 0\) 表示的曲面，若 \(F \in C^{1}\) 且 \(\nabla F \neq 0\)，则点 \(P_{0}\) 处的法向量为

\(\boldsymbol{n} = (F_{x}, F_{y}, F_{z})(P_{0})\)

切平面方程：\(F_{x}(P_{0})(x - x_{0}) + F_{y}(P_{0})(y - y_{0}) + F_{z}(P_{0})(z - z_{0}) = 0\)

法线方程：\(\displaystyle \frac{x - x_{0}}{F_{x}(P_{0})} = \frac{y - y_{0}}{F_{y}(P_{0})} = \frac{z - z_{0}}{F_{z}(P_{0})}\)
